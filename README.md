# Neptune Radar

## Download and Start Guide

Follow these steps to download the files, extract them, and start the radar.

### 1. Download the Web Radar files
- Click the Green **Code** button on the top. 
- Click **Download ZIP**.

### 2. Extract the files
- Locate the downloaded ZIP file.
- Right-click it.
- Select **Extract All...**
- Choose a folder and finish the extraction.

### 3. Start the radar
- Open the extracted folder.
- Right-click `Start-Radar.exe`
- Select **Run as Administrator**
- In Neptune under Web Radar options:
  - ip: 127.0.0.1
  - port: 7823 
  - Click Connect Webradar
  - Open your browser and go to http://127.0.0.1:7823
  
### 4. Update the radar
- Run Update-Radar.exe
